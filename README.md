Api-skeleton
============

Skeleton for API.


Requirements
============

* git bash
* python 3
* virtualenv (pip)
* virtualenvwrapper / virtualenvwrapper-win (pip)
* doit (pip)


Start
=====

* mkvirtualenv api-skeleton / workon api-skeleton
* doit install
* doit docker_build


Launch
======

* doit list
* doit launch
* doit uvicorn
* doit docker_run
